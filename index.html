Panduan Singkat Penggunaan Git Bash pada VS Code

Halaman ini menjelaskan cara menggunakan Git Bash melalui Visual Studio Code untuk melakukan push ke GitHub.

Langkah-Langkah Dasar

  1. Install Git di komputer kamu.
  2. Buka folder project menggunakan VS Code.
  3. Buka terminal di VS Code lalu pilih Git Bash.
  4. Gunakan perintah seperti:
    git init, git add ., git commit -m "pesan", dan git push.
  5. Sambungkan ke GitHub menggunakan repository yang sudah kamu buat.
